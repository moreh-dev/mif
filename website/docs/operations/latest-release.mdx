The following table lists the latest released versions of MIF components and their bundled dependencies. The MIF charts are published to the `moreh` Helm repository (`https://moreh-dev.github.io/helm-charts`).

## MIF Helm charts

| Component                | Version | Chart                            |
| :----------------------- | :------ | :------------------------------- |
| moai-inference-framework | v0.4.0  | `moreh/moai-inference-framework` |
| moai-inference-preset    | v0.5.0  | `moreh/moai-inference-preset`    |

The `moai-inference-preset` chart ships the moreh-vLLM preset (`0.15.0-260226-rc2`).

## Operators and CRDs

Installed as separate charts. See [Prerequisites](../getting-started/prerequisites.mdx) for installation order.

| Component              | Version | Chart                          |
| :--------------------- | :------ | :----------------------------- |
| Heimdall (operator)    | v0.8.0  | `moreh/heimdall`               |
| Heimdall CRD           | v0.4.0  | `moreh/heimdall-crd`           |
| Heimdall AIGateway CRD | v0.5.0  | `moreh/heimdall-aigateway-crd` |
| Odin (operator)        | v0.9.0  | `moreh/odin`                   |
| Odin CRD               | v0.9.0  | `moreh/odin-crd`               |

Odin installs LeaderWorkerSet (LWS) `v0.9.0` as a dependency. The Heimdall AIGateway gateway runtime is not a Helm chart; the Heimdall operator deploys it as a container image from an `AIGateway` resource.
